# NUTTi

[nutti.co.kr](https://nutti.co.kr) — Cafe24 기반 쇼핑몰의 스킨(테마) 소스 저장소.

## 구조

- `base/` — 원본 기준 스킨
- `skin1`, `skin3`~`skin8` — PC 스킨 버전별 소스
- `mobile/` — 모바일 스킨
- `web/` — 웹 전용 리소스 (상품 이미지 업로드 등)

각 스킨 디렉터리는 Cafe24 스킨 편집기와 동일한 구조(`layout`, `css`, `js`, `product`, `member`, `board` 등)를 따른다.

## 운영 스킨

현재 라이브로 사용 중인 스킨은 `skin8`이며, 주요 변경 사항은 `skin1`에도 동기화한다.
